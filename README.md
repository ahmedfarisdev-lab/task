# 🎯 نظام الإنتاجية | Productivity System

نظام Next.js 14 متكامل لإدارة المهام والوقت بتقنية البومودورو، مصمّم خصيصاً لتنظيم اليوم بفعالية.

## ✨ الميزات الرئيسية

- **📅 تقويم أسبوعي تفاعلي** — تنقّل بين الأيام، شاهد توزيع المهام والإنجاز بنظرة سريعة
- **✅ قائمة مهام كاملة** — إضافة، تعديل، حذف، إنجاز، تصنيف حسب الفئة
- **🍅 بومودورو متقدّم لكل مهمة** مع:
  - وقت مخصّص لكل جلسة (افتراضي 25د)
  - استراحة قصيرة (افتراضي 5د)
  - **استراحة طويلة بعد كل N جلسات (تختار العدد والمدة)**
  - انتقال تلقائي بين العمل والاستراحة
- **🔔 صوت الانتهاء + إشعارات النظام** عند انتهاء كل جلسة
- **🔊 صوت تنبيه عند بداية كل مهمة** (يمكن تفعيله/إيقافه)
- **📊 إحصائيات مباشرة** — المهام المنجزة، الجلسات المكتملة، ساعات التركيز
- **🎨 5 فئات ملوّنة**: ماستر، تصميم، عمل، إنكليزي، شخصي
- **💾 حفظ تلقائي** — جميع البيانات تُحفظ محلياً (LocalStorage)
- **📱 تصميم متجاوب** — يعمل على الموبايل والديسكتوب
- **🌐 RTL كامل** بخط Cairo + Tajawal العربيين

## 🎨 الألوان والفئات

| الفئة | اللون | الاستخدام |

| ماستر | 🟣 بنفسجي | الدراسة الأكاديمية |
| تصميم | 🟢 تركواز | التصميم والإبداع |

| إنكليزي | 🟠 برتقالي | تعلم اللغة |

## 💡 طريقة الاستخدام

1. **أضف مهمة جديدة** — اضغط زر "مهمة جديدة" واملأ التفاصيل
2. **حدّد إعدادات البومودورو** — مدة الجلسة، الاستراحة القصيرة والطويلة، كل كم جلسة استراحة طويلة
3. **اضغط Play** بجانب أي مهمة لبدء البومودورو
4. **سيشتغل المؤقت تلقائياً** ويتنقّل بين العمل والاستراحة
5. **سترى الإحصائيات تتحدّث** في الأعلى تلقائياً

## 🔧 الإعدادات

من زر الإعدادات (⚙️) يمكنك تعديل:
- تشغيل/إيقاف صوت الانتهاء
- تشغيل/إيقاف صوت بداية الجلسة
- تفعيل إشعارات النظام
- البدء التلقائي للجلسة التالية
- القيم الافتراضية لجميع المهام الجديدة
